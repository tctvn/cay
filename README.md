# Cay - Bộ gõ Telex siêu nhỏ

Bộ gõ tiếng Việt Telex siêu nhỏ gọn, dùng cho Windows.

## Tải về

Tải file [Cay.exe](https://github.com/thanhngch/Cay/releases) duy nhất (~86 KB), không cần cài đặt.

## Tính năng

- Chỉ hỗ trợ **Telex** (không VNI, không VIQR)
- Toggle bật/tắt: **Ctrl + Shift**
- Dừng gõ từ hiện tại: **Ctrl** (hoặc click chuột, phím cách, v.v.)
- Tự động khởi động cùng Windows (tùy chọn)
- Biểu tượng tray V đỏ (bật) / xám (tắt)
- Beep khi toggle

## Quy tắc Telex

| Phím | Kết quả |
|------|---------|
| aa | â |
| aw | ă |
| ee | ê |
| oo | ô |
| ow | ơ |
| uw | ư |
| dd | đ |
| z | dấu thanh (sắc, huyền, hỏi, ngã, nặng) |
| s | dấu sắc |
| f | dấu huyền |
| r | dấu hỏi |
| x | dấu ngã |
| j | dấu nặng |

## Ví dụ

- `xin chaof` → xin chào
- `aaf` → ầ
- `hoas` → hóa
- `tranj` → trận

## Yêu cầu hệ thống

- Windows 7/8/10/11 (64-bit)
- .NET Framework 4.8 (có sẵn trên Windows 10/11)
